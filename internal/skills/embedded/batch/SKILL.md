# Parallel Work Orchestration

Break complex tasks into independent parallel sub-tasks and coordinate results.

## Workflow
1. Break the task into independent sub-tasks
2. For each sub-task, spawn an agent with clear instructions
3. Each agent works independently and reports results
4. Review all results and integrate them
5. Verify the combined result is correct
