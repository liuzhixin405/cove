# Keybindings Help

## Workflow
1. Identify the keybinding or terminal behavior the user is asking about
2. Explain the relevant shortcut plainly
3. If configuration is needed, inspect the local config before editing
4. Prefer small, reversible config changes
5. Verify the updated keybinding behavior when possible
