## 🔍 关键发现

### 主流技术栈
- **Python 主导 (~66%)** — agent 的"母语"，生态最成熟
- **TypeScript/JS (~18%)** — 可视化平台、TUI 工具
- **Rust (~8%)** — 新兴趋势，注重性能和安全性 (Goose/CodeWhale)
- **Go (~4%)** — 基础设施/K8s 原生 (Kelos, Cordum)

### 架构共性
1. **ReAct Loop** — 几乎所有 harness 都使用 思考→行动→观察 循环
2. **Tool Calling** — 标准化外部工具调用接口
3. **Memory System** — 短期/长期记忆分离
4. **Planning** — 任务分解和路由
5. **Safety** — 权限控制和执行沙箱

### 开源组织
- **Anthropic** — courses, cookbooks, quickstarts（教育为主）
- **Microsoft** — AutoGen（企业级多智能体）
- **OpenAI** — Swarm（教育性轻量框架）
- **字节跳动** — deer-flow（SuperAgent）
- **NousResearch** — hermes-agent（可成长智能体）
